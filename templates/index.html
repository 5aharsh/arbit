Arbit

An API for Random Number Generation.

Arbit (lazily spelled version of Arbitrary) is a simple random number generator for number of scenarios being delivered via API.

Ratio

Generates a ratio/float number (eg. 0.8057532537386616).

Waypoint: /ratio
Method: GET
Request URL: /ratio
Response: {
            "result": 0.8057532537386616,
            "timestamp": 1577370881.1212497
        }

Generates an array of ratio (eg. 0.4060520382255244, 0.7223434561885743, 0.08803317829864132, 0.4463011129852972, 0.5861838877722955). 

Waypoint: /ratio/[size]
Method: GET
Request URL: /ratio/5
Response: {
            "result": [
                0.4060520382255244,
                0.7223434561885743,
                0.08803317829864132,
                0.4463011129852972,
                0.5861838877722955
            ],
            "timestamp": 1577371703.395178
        }

Integer

Generates a random integer between 0 to 99 (eg. 36).

Waypoint: /integer
Method: GET
Request URL: /integer
Response: {
            "result": 36,
            "timestamp": 1577372639.1929116
        }

Generates a random integer between 0 to specified maximum value (eg. 4 between 0-10).

Waypoint: /integer/[max]
Method: GET
Request URL: /integer/10
Response: {
            "result": 4,
            "timestamp": 1577372890.1929116
        }

Generates a random integer between specified minimum and maximum value (eg. 8 from range 5-10).

Waypoint: /integer/[min]/[max]
Method: GET
Request URL: /integer/5/10
Response: {
            "result": 8,
            "timestamp": 1577373019.1929116
        }

Generates an array of random integer between 0 to 99 (eg. 1, 33, 8, 6, 88).

Waypoint: /integer/n/[size]
Method: GET
Request URL: /integer/n/5
Response: {
            "result": [
                1,
                33,
                8,
                6,
                88
            ],
            "timestamp": 1577376078.8341732
        }

Generates an array of integer between 0 to specified maximum value (eg. 20,11,8,40,11 between 0-50).

Waypoint: /integer/[max]/n/[size]
Method: GET
Request URL: /integer/50/n/5
Response: {
            "result": [
                20,
                11,
                8,
                40,
                11
            ],
            "timestamp": 1577376235.2256374
        }

Generates an array of random integer between specified minimum and maximum value (eg. 8 between 5-10).

Waypoint: /integer/[min]/[max]/n/[size]
Method: GET
Request URL: /integer/5/10/n/5
Response: {
            "result": [
